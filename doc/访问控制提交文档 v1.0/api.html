权限验证接口

基本信息

Path： /api/v1/res/verify

Method： POST

接口描述：

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/json 是

Body

名称类型是否必须默认值备注其他信息
domainstring必须数据域
acc_addrstring必须被授权访问者地址
res_typestring必须资源类型
res_id []stringobject必须资源id
authnumber必须需要验证的权限

返回数据

名称类型是否必须默认值备注其他信息

warnstring非必须成功时附带的警告数据

token生成接口

基本信息

Path： /token/v1/generate

Method： POST

接口描述：

请求参数

Headers

参数名称 参数值 是否必须 示例 备注
Content-Type application/json 是

Body

名称类型是否必须默认值备注其他信息
userinfostring必须用户请求结构,如{"username":"client","pwd":"密码","orgid":"机构id"}
isencboolean必须表示userinfo是否加密
customkvobject非必须放置token中自定义的kv
audiencestring非必须请求token发起者名称,默认目录链
expiresnumber非必须过期时间,默认一天

返回数据

名称类型是否必须默认值备注其他信息
codenumber必须成功返回0 失败负值
messagestring必须成功OK 失败errMsg
dataobject非必须返回数据域　失败时为空
├─ tokenstring必须token字符串
├─ expiresnumber必须过期时间
warnstring非必须成功时警告数据
